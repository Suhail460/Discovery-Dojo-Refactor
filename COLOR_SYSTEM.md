# Color System

## Overview
A premium purple-primary palette with semantic accent colors, supporting light and dark mode through CSS custom properties.

## Primary Palette

| Token | Light | Dark | Usage |
|-------|-------|------|-------|
| `--primary` | `#6C63FF` | `#8B83FF` | Buttons, links, active states, icons |
| `--on-primary` | `#FFFFFF` | `#FFFFFF` | Text on primary backgrounds |
| `--primary-wash` | `#EEEDFF` | `#2A2745` | Subtle backgrounds, hover fills |
| `--primary-glow` | `rgba(108,99,255,0.20)` | `rgba(139,131,255,0.25)` | Box-shadow glow effects |
